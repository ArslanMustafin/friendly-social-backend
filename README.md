# Friendly Social Backend

## Description

[Nest](https://github.com/nestjs/nest)

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Envs

```shell
#.env.development
DB_USER=
DB_PASSWORD=
DB_NAME=
JWT_SECRET=
JWT_EXPIRES=
```
